Engelse tekst

Hello and welcome to beginners guide to Linux by Shadowformed Productions. If you come across this video you have probably heard Linux on the internet or from friends that use Linux but you dont exactly know what it is. Let’s assume you’re not swift with computers, when you turn on your pc chances are it’s running Windows or Mac OS X. Microsoft Windows and Mac OS X are examples of an operating system. Simply an operating system is the software that operates your computer. Windows has a taskbar which you use to run your applications, a file browser that you use to manage your files like photos and music, a web browser to access the internet and more. Linux officially known as GNU/Linux is another operating system with its own desktop, file browser and many different software applications, so the question is if Linux does everything that Windows does, then why would i use Linux as my computers operating system? There are a few key differences between Linux and other operating systems like Windows and Mac OS.

Linux is open source software which basically means that anyone can change the operating system. if someone doesn’t like the way a part of Linux works, they are able to change it to a way they like it to suit there needs. Because of this particular type of licensing, you see many different versions of Linux or distributions available. Each of them are similair but with their own benefits and all are Linux. This means that you have the freedom of choice to find a version of Linux that works best for you and believe me there is a distribution out there for everyone. While operating systems like Windows or Mac OS may suit your needs just fine. If there’s something about the way it works that bothers or annoy’s you, there is very little chance of it being changed or improved.

The second big difference is cost, currently Windows 8 Pro runs for forty dollars at it’s cheapest. Mac OS Mountain Lion runs for twenty. Both are much cheaper than they used to be, you have allways been able to download most distributions of Linux completely for free. With software improvements and bug fixes to Linux offered free as well. Application software is the same story. Almost all software for Linux is free, from office suites to web applications, games and even photo and video editors.

Another key difference is Linux offers a variety of desktop environments that you can use. Many open source developers have created intuitive desktop environments for Linux that work in their own unique way to help you better interact with your computer.

One of the most popular desktops for Linux is called Gnome, they released version 3.6 in September 2012. Which uses an overview pain by default to manage your Windows applications instead of a traditional taskbar and start menu. To open a program or switch windows you would click the ‘Activities’’ button in the top left corner or move your mouse all the way to the top left corner of the screen. Another popular desktop environment is the K desktop environment or KDE. KDE has a desktop and taskbar simillair to Winsows but is incredibly customizable Allowing you to make the desktop and taskbar exactly how you want to. You can add a number of extra featers to the taskbar and desktop with widgets. Which are customizable controls to enhance your desktop capabilities.

To summorize for many Linux means freedom, the freedom to use your computer how you want it. With his few restrictions as possible as with learning anything new. There is a learning curve but luckily there are many options for trying Linux on your pc without actually affecting your current operating system or data. Which i will include a few options in the details below. Wheter you use the computer to browse the web, perform office tasks, graphic design, games or even as a video producer like me, Linux offers an endless array of computing possibilities.

Nederlandse tekst

Hallo en welkom voor deze beginnersgids voor Linux. Als je over deze video gaat, heb je vast al gehoord van Linux, op het internet of van vrienden dat Linux gebruiken maar je weet niet exact wat het is. Laten we ervan uitgaan dat je niet vloeiend bent met computers, als je uw computer aanzet is de kans groot dat het Windows of Mac OS X draait als besturingssysteem. Een besturingssysteem is software dat je computer bestuurt. Windows heeft een taakbals waar je je applicaties op kan starten, een verkenner waar je uw bestanden zoals foto’s en muziek, het heeft ook een webbrowser om op het internet te surfen en nog vele meer. Linux officieel eigenlijk GNU/Linux is een ander besturingssysteem met z’n eigen desktop, verkenner en nog vele andere software applicaties. Dus als Linux allemaal hetzelfde doet als Windows, waarom zou ik dan Linux gebruiken als het besturingssysteem van mijn computer. Er zijn een paar belangrijke verschillen tussen Linux en andere besturingssystemen zoals Windows en Mac OS.

Linux is een geopende software wat eigenlijk wil zeggen dat iedereen het besturingssysteem kan aanpassen. Als iemand niet leuk vindt hoe dat Linux werkt, is het mogelijk om het aan te passen voor hoe zij het willen hebben. Door deze speciale manier van licenseren zie je veel verschillende versies van Linux of distributies die beschikbaar zijn. Ze zijn allemaal ongeveer gelijk aan elkaar maar met hun eigen voordelen en ze zijn allemaal Linux. Dit betekent dat je volledige vrijheid hebt voor het kiezen van een versie van Linux die het beste werkt voor jouw. Besturingssystemen zoals Windows en Mac OS kunnen voldoen aan jouw behoeften. Als er iets dat jouw stoort of verveelt, dat het verandert of verbetert wordt.

Een ander groot verschil is de prijs, Windows 8 Pro werkt voor 40 dollar op zijn goedkoopst. Mac OS Mountain voor 20 dollar. Beide zijn veel goedkoper dan dat ze vroeger zijn geweest, je hebt altijd al verschillende distributies van Linux volledig kostloos kunnen downloaden. Met software verbeteringen en bug verbeteringen door Linux aangeboden. Applicaties zijn hetzelfde verhaal. Bijna alle software in Linux is compleet vrij, van kantoortoepassingen tot web applicaties, spelletjes en zelfs foto en video editors.
