Softwareontwikkeling & -pakketten

Taak 06: Vergelijken van een workflow versie 1

Toen we deze opdracht kregen wist ik niet goed welke staticgen ik moest nemen. Dus heb ik wat rondgevraagd en ben ik aan deze taak begonnen. Om te beginnen moest ik hexo installeren maar dat ging niet meteen. Eerst moest ik chocolaty installeren zodat ik npm kon installeren. Nadat ik chocolaty had geïnstalleerd moest ik dit in mijn path krijgen want anders werkte het niet. Dit is me dan gelukt en zo heb ik npm kunnen installeren. Dan kon ik verdergaan en heb ik hexo geïnstalleerd.

Hexo is eigenlijk een blog dus was het ietsje anders om er toch een website van te kunnen maken. Door in de opdrachtprompt te typen “hexo init “hier een projectnaam”” maak je een nieuw project. In dit project moet je dan posts en pages maken. Posts kan je maken door in de opdarchtprompt “hexo new “hier de naam van een post””, deze verschijnt dan in de map _posts. Om een page te maken moet je “page” achter het woordje “new” zetten (hexo new page “hier de naam van een pagina”), dan wordt er een nieuwe map aangemaakt met de naam van een pagina. Als je dit hebt gedaan ben je al een hele stap verder.

Dan moest ik zien te vinden hoe ik verschillende pagina’s kon maken en ze kon linken. Dit moest ik doen door in de _config.yml van mijn landcape iets aan te passen. Zo kan je het aan elkaar linken aan de hand van de pages. Om tekst toe te voegen aan een post moet je in het .md bestand je tekst typen in markdown. Bij de page moet je hetzelfde doen.

Dan heb ik een beetje veranderd aan de opmaak van de webiste. Dit heb ik gedaan door in de map css, in de bestanden die daarin staan te wijzigen.

Dit heeft wel een beetje tijd gevergd maar ik ben blij dat ik bij dit eindresultaat ben uitgekomen.
